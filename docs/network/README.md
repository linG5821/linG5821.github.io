### 字节序

#### 大端小端字节序

```
自然顺序
4字节数组(地址增序为低到高): 低 0 1 2 3 高
数字32位(数字的二进制左高右低): 高2(◻◻◻◻◻◻◻◻) 高1(◻◻◻◻◻◻◻◻) 低1(◻◻◻◻◻◻◻◻) 低2(◻◻◻◻◻◻◻◻)

大端(高位对应低地址)
字节数组 0   1    2    3
       高2  高1  低1  低2
       
小端(低地址对低位，高地址对高位)
字节数组 0   1    2    3
       低2  低1  高1  高2


例如 1a1b1c1d 这个数字
1a1b 属于高8位
1c1d 属于低8位

高位字节---->低位字节 
1a  1b  1c  1d
------------->
低地址------->高地址 ---------->大端模式
------------->

低位字节---->高位字节
1d  1c  1b  1a
------------->
低地址------->高地址 ---------->小端模式
------------->
```
