    # 大端小端字节序

    例如 1a1b1c1d 这个数字
1a1b 属于高8位
1c1d 属于低8位

高位字节---->低位字节 
1a  1b  1c  1d
------------->
低地址------->高地址 ---------->大端模式
------------->

低位字节---->高位字节
1d  1c  1b  1a
------------->
低地址------->高地址 ---------->小端模式
------------->
